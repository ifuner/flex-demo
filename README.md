# flex-demo

> 本项目由@baoji/fie-toolkit-vue套件创建

## 安装FIE

```bash
 cnpm install -g @baoji/fie 
```

## 安装套件 

```bash
 fie install toolkit-vue
```

## 更新套件

```bash
 fie update @baoji/fie-toolkit-vue
```

## 套件命令

### 初始化项目

```bash
 fie init vue 
```

### 添加页面

```bash
 fie page
```

### 本地开发

```bash
 fie start
```

### 本地构建

```bash
 fie build
```

### 代码发布

```bash
 fie publish
```
